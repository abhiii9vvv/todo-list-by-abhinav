# To-do List by Abhinav 📝

A modern, stylish, and interactive To-do List web app built using HTML, CSS, and JavaScript. This app includes features like task editing, filtering, searching, and local storage support — all wrapped in a visually appealing VHS-style theme.

## 🚀 Features

- ✅ Add, Edit, Complete, and Delete tasks
- 🔍 Real-time task search
- 📂 Filter tasks (All / Active / Completed)
- 💾 Persistent storage via LocalStorage
- 🎨 Animated VHS UI with glitch effects
