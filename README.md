# ToDoFlow

**ToDoFlow** is an advanced, user-friendly task management web application designed to help you organize your tasks efficiently. It offers task categorization, priority levels, due dates, and completion tracking with a clean, intuitive interface.

---

## Features

- Add, edit, and delete tasks easily.
- Categorize tasks into **Work**, **Personal**, **Study**, **Health**, and **Other**.
- Set task priority levels: **Low**, **Medium**, and **High**.
- Assign due dates to tasks.
- Mark tasks as completed with a visual toggle.
- Filter tasks by category, priority, date, and status.
- Search tasks with a convenient search bar.
- Responsive design for desktop and mobile.
- Integrated social links with GitHub and LinkedIn icons.

---

## Technologies Used

- **HTML5** — Markup structure.
- **CSS3** — Styling and responsive design.
- **JavaScript (ES6+)** — Dynamic task handling and UI updates.
- **Font Awesome** — Icon library.
- **Google Fonts (Inter)** — Typography.
